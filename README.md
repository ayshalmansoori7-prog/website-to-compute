## How to Play

1. Enter team names
2. Select 1-6 categories
3. Click on a category to see the question
4. Answer before time runs out
5. Compete with the other team
6. View final results and winner

## Game Rules

- Each team takes turns answering questions
- Correct answers earn points
- 60-second timer per question
- Teams alternate after each question
- Game ends when all questions are answered
- Winner is the team with the most points
