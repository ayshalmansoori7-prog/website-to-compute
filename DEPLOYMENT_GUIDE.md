## **Option 1: GitHub Pages (Recommended - Easiest)**

### **Steps:**

1. Go to your GitHub repository: `https://github.com/ayshalmansoori7-prog/website-to-compute`

2. Click **Settings** (top right)

3. Scroll down and find **Pages** section on the left

4. Under "Source", select **Deploy from a branch**

5. Select **Branch: main** and **/(root)**

6. Click **Save**

7. Wait 1-2 minutes, then your game will be live at:
   ```
   https://ayshalmansoori7-prog.github.io/website-to-compute/
   ```

That's it! Share this link and open in any browser.

## **Option 2: Vercel (Even Easier - Auto Deploy)**

1. Go to `https://vercel.com`
2. Sign up with GitHub
3. Click "New Project"
4. Select your `website-to-compute` repository
5. Click "Deploy"
6. Your game will be live in 30 seconds at a URL like:
   ```
   https://website-to-compute.vercel.app
   ```

---

## **Option 3: Netlify (Very Easy)**

1. Go to `https://netlify.com`
2. Click "Sign up" and choose GitHub
3. Click "New site from Git"
4. Select your `website-to-compute` repository
5. Click "Deploy site"
6. Your game will be live at:
   ```
   https://[random-name].netlify.app
   ```

## **Troubleshooting**

**Pages not showing?**
- Make sure `index.html` or `standalone-game.html` is in the root directory
- Wait 5 minutes for GitHub Pages to build
- Check that Pages is enabled in repository Settings

**Games not loading?**
- Clear your browser cache (Cmd+Shift+R on Mac, Ctrl+Shift+R on Windows)
- Try a different browser
- Check console for errors (F12)
